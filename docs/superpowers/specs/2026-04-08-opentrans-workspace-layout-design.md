# OpenTrans 会话恢复与分栏调整设计

## 目标

1. 关闭应用后再次打开时，自动恢复上一次成功打开的项目。
2. 如果上次项目路径已失效，启动时静默回到空状态，不弹错误。
3. 原文与译文面板支持拖拽调整宽度，并持久化到本地。

## 设计

### 项目恢复

- 在渲染层本地持久化最近一次成功打开的工作区：
  - `srcDir`
  - `translatorDir`
- 应用启动时自动读取该状态并尝试调用 `get-file-tree` 恢复。
- 若恢复失败，则清空会话并保持空状态。

### 分栏调整

- 左侧文件树宽度保持固定。
- 原文与译文之间增加拖拽分隔条。
- 分栏比例仅控制原文/译文区域，默认 50:50。
- 比例限制在 30% 到 70% 之间，避免某一侧被拖没。
- 当前比例持久化到本地，下次启动继续使用。
